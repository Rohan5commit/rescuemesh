# Fire Safety Knowledge Pack

Pre-built knowledge pack for fire and smoke emergencies.

## Contents

- Fire Emergency Response SOP
- Smoke Inhalation First Aid
- Building Evacuation Procedures
- Fire Extinguisher Usage Guide

## Usage

This pack is automatically loaded when creating a new incident case.
The contents are ingested into the local RAG system for retrieval.

## Source

Based on standard fire safety protocols and first-response guidelines.
